
Log file parser to count the webpage visit using Map Reduce 
 - Implemented this after completing 'Introduction to Hadoop and MapReduce' course from Udacity.

Dataset 
 - Download data from http://content.udacity-data.com/course/hadoop/forum_data.tar.gz
 - unzip file and copy access_log to data folder in this code. 

System used:
 - Cloudera-vm-4.7.0.0 (http://www.cloudera.com/content/support/en/downloads/quickstart_vms/cdh-4-7-x.html) 

Run code
 - #sh run.sh
